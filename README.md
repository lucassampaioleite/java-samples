# java-samples

## Objetivo
- Este repositório tem o objetivo de agrupar códigos-exemplos utilizados em aulas de disciplinas que envolvem a linguagem de programação Java.

3. [Exemplo de tratamento de exceções em um sistema de reservas de quartos.](./ExceptionHandling)
4. [Exemplo de aplicação de Generics e da interface Comparable.](./GenericsAndComparable)
5. [Demonstração dos conceitos de Interfaces e Services](./InterfacesAndServices)
6. [Exemplos de aplicação de Java Lambda Expressions](./LambdaExpressions)
7. [Leitura de arquivos de texto utilizando as classes File, BufferedReader e try-catch-resources.](./ReadFile)
8. [Escrita de arquivos de texto utilizando BufferedWriter e try-catch-resources.](./WriteFile)
9. [Implementação de uma Lista a partir de um vetor.](./ListWithArray)
